## Overview

**Natural WanderLust (NWL)** is a comprehensive travel website designed to enhance user experiences by simplifying content management and facilitating trip planning. Our goal is to create an informative and aesthetically pleasing platform that appeals to both foreign and local travelers.

## Objectives

1. **Enhance User Experience**: Provide a seamless and intuitive interface for users to explore travel destinations, plan their trips, and access necessary information.
2. **Simplify Content Management**: Develop an easy-to-use content management system for efficient handling and updating of travel information.
3. **Facilitate Trip Planning**: Offer tools and resources to help users in planning their trips, including itineraries, recommendations, and booking options.

## Target Users

- **Foreigners and Locals**: Our platform is designed for both foreign and local travelers who are interested in exploring various travel destinations.
- **Age 25 and Above**: The website caters to an audience aged 25 and above, focusing on those with a keen interest in travel.

## Themes

Our website features nature-inspired themes with earthy tones and calming elements. This design choice aims to create a relaxing and inviting atmosphere for users, enhancing their overall browsing experience.

## Features

- **User-Friendly Interface**: Intuitive navigation and responsive design to cater to various devices.
- **Content Management System**: Simplified backend for easy content updates and management.
- **Trip Planning Tools**: Comprehensive tools for itinerary planning, recommendations, and booking assistance.
- **Rich Content**: Detailed information on travel destinations, attractions, accommodations, and more.

## Development Setup

### Prerequisites

Web Server: Apache or Nginx.
PHP: Version 7.4 or higher.
MySQL: Version 5.7 or higher.

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/dylaascreate/wanderlust.git
   ```
2. Navigate to the project directory:
   ```sh
   cd wanderlust
   ```
